## 🚀 v0.4.13 — 当前最新版本

**发布日期**: 2026-06-07

### 本次修复与改进

#### 🐛 Bug 修复
1. **护盾状态 (`shieldActive`) 同步修复** — 答错、切换难度/模式时现在会正确重置护盾状态，避免连胜中断后 UI 仍显示护盾激活。
2. **计时挑战结束后仍可提交** — 时间耗尽 (`timeLeft <= 0`) 后，输入框自动禁用，提交/输入/点击牌面等操作均会被阻止。
3. **发牌等待期间状态竞争修复** — 跳过或答对后等待新牌的 0.6~1.5 秒内，输入、提交、提示、显示答案、切换难度等操作现在会被忽略，避免旧题目的操作污染新牌状态。
4. **错题复习模式为空时不生成新牌** — 点击"错题复习"但错题本为空时，现在会正确切回经典模式并立即生成一组新牌。
5. **`loadFromStorage` Port 冗余调用清理** — JavaScript 端已主动读取本地存储，移除 Elm 中无效的重复触发。
6. **新局时输入提示不重置** — `inputHint` 与 `liveResult` 在发新牌时现在会正确清空。
7. **"全部解法"按钮在无解题目上的反馈** — 无解题目点击"全部"时现在显示"这道题没有解法，请跳过换一组"。
8. **"步骤"按钮在无解题目上的反馈** — 无解题目点击"步骤"时现在显示"这道题没有解法，无法显示步骤"。

#### 🎨 UX 改进
9. **退格键改为按字符删除** — 之前退格会一次性删除整个 token（如 `12` 整个删除），现在改为逐个字符删除，更符合用户预期。
10. **虚拟键盘 `(` 不再自动补全 `)`** — 自动补全容易导致括号不平衡，现在改为只插入一个 `(`，由用户自主管理括号。
11. **分享链接附带题目 Hash** — 分享文本现在包含 `#1,2,3,4` 格式的 URL Hash，接收方打开链接即可直接看到相同题目。
12. **牌面悬停提示显示牌面字符** — A/J/Q/K 的悬停提示现在显示 `A`/`J`/`Q`/`K`，而不是数字 `1`/`11`/`12`/`13`。
13. **主题按钮文案中文化** — 主题切换按钮由英文 "Dark/Light" 改为中文 "深色/浅色"。

#### 🔧 维护
14. **全站版本号统一** — README、HTML、Manifest、Footer 全部统一为 **v0.4.13**，解决此前四个文件版本不一致的问题。
15. **Service Worker 缓存版本升级** — 从 `v15` 升级到 `v16`，确保已安装 PWA 用户能获取到最新脚本。
16. **新增 `CHANGELOG.md`** — 建立版本公告与历史记录文档。
